# Memoteca

Projeto desenvolvido durante o curso de **Angular 14** da Alura. Primeiro projeto utilizando angular, usando o projeto como material de estudo para aprender alguns conceitos do framework -> componentização, serviços, rotas e consumo de dados.

# Objetivo 

Praticar os conceitos iniciais do Angular atráves da construção de uma aplicação simples, acompanhando os estudos e experimentação durante o curso.

# Tecnologias
- Angular 14
- Typescript
- Html
- CSS
